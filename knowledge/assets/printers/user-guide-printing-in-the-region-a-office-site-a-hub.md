# **User Guide - How to Setup and Configure a printer (Mac & Windows)**

### **🍎Mac laptop Users**

**1️⃣ Add Printer via Self Service** 🔹 Open **Spotlight** (Cmd + Space) → search **Self Service** 🔹 In **Self Service** , search for **HYD Printers** 🔹 Click **Install/Reinstall** ✅ Once installed, your printer will be ready to use

### **💻 Windows Users**

**1️⃣ Open Printer Settings** ⚙️ Press **Windows + I** → search **Printers & Scanners**

**2️⃣ Add Printer** 🔹 If your printer is listed → click **Add Device** 🔹 If it isn’t listed → select **The printer I want isn’t listed** → **Add manually** 🔹 Choose **Add a printer using a TCP/IP address or hostname**

**🌐 Enter the correct IP:**

- 🏢 **10th Floor:** <PHONE_NUMBER>
- 🏢 **7th Floor:** <PHONE_NUMBER>

➡️ Click **Next** and follow the prompts to complete the installation

## **General Tips**

- Ensure your device is connected to the same network as the printer(In-office).
- If prompted, Windows users please install any drivers (macOS usually handles this automatically).
- Please feel free to reach out to me or Sneha for any doubts or setup issues.

## **Pro Tip**

Disconnect VPN while you are printing a document.
